# Tarea-8



Programa que implementa una lista doblemente enlazada usando memoria dinámica (new y delete) y punteros.

Incluye dos versiones:

Versión 1: lista con head
Versión 2: lista optimizada con head y tail

Funciones implementadas:

Insertar al inicio
Insertar al final
Mostrar hacia adelante
Mostrar hacia atrás
Buscar nodos
Eliminar nodos
Contar nodos
Liberar memoria
